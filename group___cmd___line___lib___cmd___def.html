Detailed Description

Module for defining lists of commands.

Command

Command is defined by the cmd_desc_t data type. Every command is identified by its unique name (cmd_desc_st::name).

Command has a brief one line description (cmd_desc_st::one_line) and a detailed description (cmd_desc_st::description)

The routine done by the command is defined as a function with the cmd_do_fcn_t notation.

List of commands

Commands are grouped together into lists. The List is an array of commands, terminated by CMD_LINE_END_OF_LIST.

Lists can be nested by CMD_LINE_INCLUDE_SUBLIST macro. The maximum depth is defined by CMD_LINE_MAX_INDENT.

Basic commands

The Command line module is shipped with a basic set of commands defined in basic_cmd_list. Feel free to use this list as the root list for cmd_line_init or to include it as a sublist into your own commands list.
